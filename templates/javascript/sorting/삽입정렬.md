- 삽입 정렬이란 각 수자를 적절한 위치에 삽입하는 정렬 기법이다.

1. 각 단계세서 현재 원소가 삽입될 위치를 찾는다.
2. 적절한 위치에 도달할 때까지 반복적으로 왼쪽으로 이동한다.

```
function insertionSort(arr) {
  for(let i = 1; i < arr.length; i++) {
    for (let j = i; j >0; j++) {
      // 인덱스 j부터 i까지 1씩 감소하며 반복
      if(arr[j] < arr[j-1]) {
        // 한 칸씩 왼쪽으로 이동
        // 스와프(swap)
        let temp = arr[j];
        arr[j] = arr[j-1];
        arr[j-1] = temp;
      } else {
        // 자기보다 작은 데이터를 만나면 그 위치에서 멈춤
          break;
      }
    }
  }
}
```
